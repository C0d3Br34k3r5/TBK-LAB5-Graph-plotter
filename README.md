# CVUT FEL TBK LAB5 Graph plotter
**Required packages:**
+ matplotlib
+ os
+ sys

+ math

**Graphs can be saved as:**
+ PNG
+ SVG
+ NONE (view only)

## Plotter of |s11| and VSWR
**Required files:**
+ Table of s11 params *(exported from VNA in TXT, with removed header, 3 columns in order FREQ – MAG – ANG) (see attached version)*  

**For help message use:**
```
python auto_graph.py -h
```
Which will generate:

```
usage: auto_graph.py [-h] -v VERSION -f FILE_PATH [-s PRINT_PATH] [-t TITLE] [-c COLOR] [-b]

A script to plot graph of s11 or VSWR parameters from messy pile of numbers in *.txt from TBK; CTU FEE

options:
  -h, --help            show this help message and exit
  -v, --v VERSION       type of graph plotted, use:'1', 'A', 'S' or 'S11' for s11 graph and '2', 'B', 'V' or 'VSWR' for VSWR graph
  -f, --f FILE_PATH     path to *.txt file with s11 params (downloaded from Moodle)
  -s, --s PRINT_PATH    saves graph as image to PRINT_PATH, filename can be specified as *.png (default) or *.svg
  -t, --t TITLE         title of plotted graph
  -c, --c COLOR         set color of graph, use: 'blue','green','red','yellow','magenta', 'cyan', 'black' and 'white' or color names defined by CSS
  -b, --b               blind mode, does not show loaded data and does not show graph after save, disabling show is ignored if PRINT_PATH is not specified
```
**Minimal arguments example _(with save, disabling debug print)_:**
```
python auto_graph.py -f "PATH_TO_S11_PARAMS" -v 1 -s "PATH_TO_PRINT_DIRECTORY" -b
```
IRL:
```
python auto_graph.py -v 1 -f "s11_params.txt" -b -s "./"
```

